# E-Learning Platform (Full Stack)

## 🚀 Technologies Used
- ASP.NET Core Web API
- SQL Server
- HTML, CSS, JavaScript
- Bootstrap

## 📌 Features
- User Registration
- Course Management
- Quiz System
- Score Tracking
- Dashboard with Progress

## 🔄 Workflow
User → Course → Lesson → Quiz → Question → Submit → Result

## ▶️ How to Run

### Backend
cd Backend
dotnet run

### Frontend
Open index.html using Live Server

## 📊 API Testing
Use Swagger UI to test APIs
